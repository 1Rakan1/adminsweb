# نظام ساعدني

نظام إدارة المشاكل الفنية مع نظام رتب متكامل

## الميزات

- تسجيل دخول/تسجيل مستخدمين
- 3 مستويات للمشاكل (منخفض/متوسط/عالي)
- نظام رتب (قائد/مساعد/أدمن/مستخدم)
- دردشة مباشرة لحل المشاكل
