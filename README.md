# نظام ساعدني للدعم الفني

## الميزات الرئيسية
- نظام مصادقة كامل
- إدارة رتب (قائد، مساعد، أدمن، مستخدم)
- تقديم مشاكل فنية بمستويات مختلفة
- نظام دردشة مباشرة
- تكامل مع ديسكورد للتواصل الصوتي

## كيفية التشغيل

### متطلبات النظام
- Node.js 14+
- MongoDB 4.4+

### خطوات التنصيب
1. تشغيل قاعدة البيانات:
```bash
mongod
```

2. تشغيل الخادم:
```bash
cd server
npm install
npm start
```

3. تشغيل الواجهة:
```bash
cd client
npm install
npm start
```

## زر الديسكورد
متوفر في جميع صفحات النظام للتواصل الصوتي المباشر
